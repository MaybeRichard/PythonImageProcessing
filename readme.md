# Python 计算机视觉 图像处理


### 这个仓库包含了《Python 计算机视觉》这本书的示例代码，用于图像处理和计算机视觉相关的任务。我们将演示如何使用 Python 以及一些流行的库，如 OpenCV 和 PIL，来处理和分析图像。

## 内容

- 图像读取与显示
- 基本图像处理（调整大小、旋转、裁剪等）
- 图像滤波与边缘检测
- 图像分割与特征提取
- 图像识别与分类

## 环境要求

- Python 3.6 或更高版本
- Opencv
- PIL
- Numpy
